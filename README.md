# 📝 TaskFlow - TODO Application

A modern, responsive web-based task manager built with HTML, CSS, and JavaScript.

---


## ✨ Features

- **Task Creation** — Add new tasks instantly
- **Task Completion** — Mark tasks done with visual feedback
- **Task Deletion** — Remove tasks with one click
- **Data Persistence** — Tasks saved via local storage
- **Responsive Design** — Works on desktop, tablet, and mobile

---

## 🚀 Getting Started

1. Clone or download the project
2. Open `index.html` in any web browser
3. Start managing your tasks!

## 💻 Usage

| Action | How To |
|--------|--------|
| Add a task | Type in the input field and press Enter or click Add |
| Complete a task | Click on the task to mark it as done |
| Delete a task | Click the delete button next to the task |
| Data saved | Tasks persist automatically between sessions |
